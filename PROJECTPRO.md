# Project Knowledge Template

<!--
Use this file to write a knowledge document for one project.

Good uses:
- Plans to renovate a kitchen.
- Background on your business.
- A research project.
- A community event.
- A product idea.
- A personal admin project.

The goal is to give an agent enough context to help you think, plan, write,
research, remember details, and ask better questions.

Keep this file factual and useful. It does not need to be perfect. Short notes
are better than empty sections.

Do not store passwords, API keys, bank details, private medical records, legal
instructions, identity documents, confidential client data, or anything that
belongs in a secure system.
-->
